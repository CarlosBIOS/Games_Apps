HTML(HyperText Markup Language) serve para o conteúdo, como fosse a base do site. Define a estrutura do conteúdo: 
títulos, parágrafos, imagens, links, etc

CSS(Cascading Style Sheets) serve para a parte estética do site!! Portanto, se vir que um site tem os atributos de
HTML em relação à estética, o site está mal feito, pois esses mesmo atributos já são considerados como
lixo(<center> ou <font>, <border>, entre outros). Controla cores, fontes, tamanhos, layout, posicionamento, etc.
Existe 3 maneiras de definir o CSS: Inline(Vamos usá-la quando quero especificar só num elemento), Internal(Vou 
usá-la quando quero especificar só num Webpage) e External(Quando quero que tenha em mais do que 1 webpage, portanto
vou ter que criar um ficheiro à parte, por exemplo: style.css).
