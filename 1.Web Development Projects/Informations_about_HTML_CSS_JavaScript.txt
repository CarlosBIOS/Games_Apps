HTML(HyperText Markup Language) serve para o conteúdo, como fosse a base do site. Define a estrutura do conteúdo: 
títulos, parágrafos, imagens, links, etc

CSS(Cascading Style Sheets) serve para a parte estética do site!! Portanto, se vir que um site tem os atributos de
HTML em relação à estética, o site está mal feito, pois esses mesmo atributos já são considerados como
lixo(<center> ou <font>, <border>, entre outros). Controla cores, fontes, tamanhos, layout, posicionamento, etc.
Existe 3 maneiras de definir o CSS: Inline(Vamos usá-la quando quero especificar só num elemento), Internal(Vou 
usá-la quando quero especificar só num Webpage) e External(Quando quero que tenha em mais do que 1 webpage, portanto
vou ter que criar um ficheiro à parte, por exemplo: style.css). ~
Todos os selectors que existe em CSS: 
Element selector, Class selector, Id selector, Atribute selector e Universal selector!!!
Id selector(#word) vs class selector(.word): class selector pode ser usado várias vezes no mesmo documento, enquanto
o Id só pode ser usado 1 vez!!

Font Size: 1px(pixel): 1/96th inch(0.26 mm); 1 pt(point, e é usado no Word): 0.72th inch(0.35 mm); 1em(basicamente, se
definir 1em, numa tag e no body tem 20px, então o tamanho dessa tag vai ser 1*20=20px): 100% of parent; 1rem(Se definir
no head x px, então vai manter esse x pix, mesmo que tenha alterado ao longo do body): 100% root(recomendam usar este)

O JavaScript permite que o site tenha interatividade, dinamismo e funcionalidade. Permite que elementos respondam a 
ações do usuário, como cliques, digitação e movimento do mouse.
